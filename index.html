⚙️ Supabase DB 연결 설정
📌 Supabase를 연결하면 여러 컴퓨터에서 실시간으로 같은 데이터를 볼 수 있습니다.
무료 계정: supabase.com 에서 생성 후 아래 정보를 입력하세요.
Settings → API → Project URL
Settings → API → Project API Keys → anon public
📋 Supabase SQL Editor에서 아래 테이블을 생성해주세요:
create table items (id bigserial primary key, platform text, ship_date text, code text, name text, qty int, expiry text, suffix text, location text default '', updated_at timestamptz default now());
create table platforms (id bigserial primary key, name text unique, color text, sort_order int default 0, locations text default '[]', loc_cols text default '{}');
create table logs (id bigserial primary key, type text, msg text, created_at timestamptz default now());
⚠️ 기존 platforms 테이블이 있다면 아래 SQL로 컬럼 추가:
alter table platforms add column if not exists sort_order int default 0;
alter table platforms add column if not exists locations text default '[]';
alter table platforms add column if not exists loc_cols text default '{}';
